# Focus-Board-Chrome-Extension

Turn every new tab into your personal command center.

This Chrome extension replaces the default new tab page with a minimal, distraction-free productivity dashboard that helps you stay focused, organized, and intentional throughout the day.

---

## ✨ Features

### 🧠 Daily Focus System
- Get prompted every morning:
  > *“What is your main focus today?”*
- Your focus stays visible all day
- Automatically resets the next day

---

### 🕒 Live Dashboard
- Personalized greeting (with your name)
- Real-time clock
- Full-screen wallpaper (custom upload supported)

---

### ✅ Task Management
- Add tasks instantly
- Mark tasks as complete
- Delete tasks easily
- Persistent across sessions

---

### 🚫 Focus Mode (Distraction Blocker)
- Toggle Focus Mode ON/OFF
- Blocks distracting websites like:
  - Instagram
  - YouTube
  - LinkedIn
- Instead of an error page, shows your **focus message**

---

### ⚙️ Custom Settings
- Update your name
- Upload/change wallpaper
- Manage blocked websites
- Clean and simple settings panel

## 🛠️ Tech Stack

- Chrome Extension (Manifest V3)
- HTML, CSS, JavaScript
- `chrome.storage.local` for persistence
- Background service worker for Focus Mode blocking

---

## 📦 Installation

### Option 1: Load Locally (Developer Mode)

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git

2. Open Chrome and go to:
     chrome://extensions/

3. Enable Developer Mode (top right)
4. Click Load unpacked
5. Select the project folder
6. Open a new tab
